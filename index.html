在地人私藏版

馬祖導覽手冊

南竿 · 北竿 · 東引　｜　美食 × 景點

南竿 北竿 東引

南竿

馬祖列島的行政與交通中心，
老街聚落、坑道與夕陽並存。

美食
涼山頂
在地人愛
+
觀光客鮮少知道，但在地人請客幾乎首選這裡。份量偏大，四人前往種類可能吃不了太多，建議人數多一點再來。
儷儐餐廳
觀光推薦 在地人愛
+
觀光客與在地人都推，價格平實，連阿兵哥也是常客。水餃是招牌，份量適中，四人吃剛剛好。
香林小館
台灣500碗 在地人愛
+
老夫妻經營，口味道地實惠，今年入選第二屆台灣500碗小吃。推薦：鼎邊糊、老酒麵線、紅糟炒飯。
🥟 極少數有販賣「蒸」地瓜餃的店家，不容錯過！
萬家香
僅早午餐
+
狗麵與餛飩聞名，散客與在地人都愛。只開早上到中午，適合麵條配湯再點滷味，當作豐盛早午餐。
獅子市場早餐（三選）
強烈推薦
+
早餐一定要到市場報到一次！推薦三攤：

🥚 陳阿伯蛋餅：外皮酥脆，近似蔥油餅口感，是在地共同早餐記憶。

🍤 虫弟餅（面對陳阿伯蛋餅的右後方）：外型像本島蚵嗲，現炸。
點餐時記得說「辣椒包裡面」，就能得到內餡更夠味的版本——觀光客幾乎不知道的吃法！
🐟 阿妹鼎邊糊：觀光客愛店，海鮮用料豐富，喜歡鮮魚湯底的必點。
歡歡冰沙
需先探勘
+
大推花生與綠豆冰沙，口感極為綿密。位在仁愛村，建議前往前先確認是否有開店。
景點
媽祖巨神像 × 四維夕陽
私藏景點
+
兩個景點相鄰，建議排在傍晚一起走。媽祖神像上方視野極佳，四維夕陽是在地人私藏、觀光客幾乎不會查到的低調美景。
馬港天后宮
信仰中心
+
馬祖最重要的信仰核心，傳說媽祖娘娘當初遺體漂至此地，許多人特地來此參拜。
津沙聚落
打卡
+
保存完整的閩東式石屋聚落，白天漫步其中有種悠閒的 Chill 感，光影與石牆都是好畫面。
北海坑道 × 大漢據點
必訪
+
白天走坑道四周步道，感受壯觀的地下水道；晚上有夜間搖櫓行程，在坑道內欣賞藍眼淚。
坑道隔壁的大漢據點同樣值得進去走走，不要錯過。
鐵堡 × 津仁步道
散步
+
離北海坑道不遠，津仁步道沿線可邊走邊看海景，節奏輕鬆。
珠螺國小（戰地防空洞）
需預約
+
舊戰地小學，樓上是教室，地下可參觀防空洞，歷史感濃厚。
預約參觀請洽粉專：facebook.com/MYDA.com.tw
八八坑道
順遊
+
馬祖高粱酒的釀造坑道，地方不大，可順道進去體驗。
枕戈待旦牌坊
拍照
+
位於福澳村附近，許多觀光客在牌坊下留影，眺望出去的海景相當漂亮。
津沙 or 仁愛沙灘
踏浪
+
南竿最乾淨、最寬廣的沙灘，非常適合下去走走踏浪，天氣好時絕對值得一逛。

北竿

閩東聚落保存最完整的島嶼，
芹壁石屋、沙灘、梅花鹿一應俱全。

美食
鏡沃
朋友公認第一
+
朋友們公認馬祖最好吃的老酒麵線就在這裡。
饌食
在地人愛
+
牆上滿是觀光客與阿兵哥簽名留言的在地老店，料理樣樣好吃，去北竿必吃。
家適
下午茶
+
許多人在這裡點鬆餅當下午茶，順便欣賞芹壁一帶的海景，氣氛悠閒。
芹沃
觀光推薦
+
披薩是招牌，帶台灣朋友去吃都說值得。份量挺大，建議人多再去。
金魚水手作坊
特色點心
+
販賣馬祖傳統小點：糯米時，以及一種外形像湯圓、馬祖話叫做「系」的點心，朋友強推，值得一試。
阿婆魚麵
名店
+
吃魚麵的首選名店，許多朋友一致推薦。湯麵與炒麵都有，怕腥的建議選炒的。
忠漁原味
特色體驗
+
主打海鮮炭烤，最特別的是喝酒用碗上桌，豪邁風格十足，體驗感很強。
景點
戰爭和平紀念公園主題館
歷史
+
館內陳設大量兩岸戰爭時期的歷史舊物，是了解馬祖軍事歷史的最佳窗口。
螺山步道
私藏必去
+
個人大推！在地人覺得比芹壁更值得的景點，觀光客幾乎不會去。保有最原始的步道風貌，值得特地排進行程。
芹壁
打卡
+
馬祖閩東式建築保存最完善的區域，石屋錯落有致，許多人在此吃鬆餅、看夕陽。近年逐漸商業化，但風景依然值得。
坂里沙灘
最乾淨
+
印象中全馬祖最乾淨的沙灘，在地學生最喜歡來這裡玩耍，沙質細白。
短坡觀景平台
新景點
+
近年新整理的小平台，爬上去可遠眺整個北竿，也能觀看北竿機場的飛機起降，角度獨特。
壁山
全島制高點
+
馬祖最高山，登頂可俯瞰整個北竿島嶼全景，天氣好時視野壯闊。
大坵島
絕對必去
+
個人認為北竿最不能錯過的行程！整座島只有一戶人家，原生自然景色完整保留，島上還有一群梅花鹿自由活動。
🦌 上船前可向船家購買樹葉帶上島，近距離餵食梅花鹿，超療癒！

東引

最遠、最靜，
燈塔、海景與慢活是這裡的日常。

美食
莒興雞排
強推
+
人氣超旺的碳烤雞排，許多人慕名而來。
⚠️ 建議下午就先電話預訂，阿兵哥有時候會包場訂光，晚到就沒了！
泰利食堂
無菜單料理
+
年輕人返鄉創業開的無菜單料理，許多海鮮都是老闆親自釣上來的，新鮮度有保障，充滿在地情懷。
國利豆花
越賣越有名
+
本店在莒光，東引分店一樣好吃。本業是做豆腐，但豆花越來越出名，連南竿人到莒光或東引都會特地帶回家。
景點
東引燈塔
必訪
+
來東引幾乎所有人都是為了這座燈塔而來，搭配遼闊海景一起構圖，是東引最具代表性的畫面。
安東坑道
必踩
+
到東引幾乎人人都會去的地下坑道，規模壯觀，是感受馬祖軍事歷史的重要一站。
海上看東引行程
期間限定需預約
+
在地人極力推薦、學校校外教學必排的行程，從海上的視角欣賞東引島嶼全貌，視覺震撼。
📞 需事先預約，詳情參考：matsu-news.gov.tw
鹹味島合作社
慢活咖啡
+
在地人經營、質感十足的小店，坐在裡面喝咖啡看海景，體驗東引最純粹的慢活步調。
